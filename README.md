# 公文一键排版工具

本仓库提供一个本地浏览器运行的公文排版工具，支持上传 Word/TXT 文件或直接粘贴文本，完成格式清洗、段落类型确认和 DOCX 导出。

## 本次更新

- 新增粘贴文本排版入口，可直接粘贴正文片段后解析、预览并下载 DOCX。
- 支持无主标题的正文片段模式，主标题可留空，不再阻塞预览和导出。
- 新增上行文版式选项，支持填写发文字号和签发人，并在预览和导出的 Word 中生成对应版头行。
- 预览编辑栏新增副标题、称呼、结语、落款、日期等手动段落类型。
- 修复序号纠偏统计一直为 0 的问题。
- 修复中文编号超过十时二级标题可能出现 `undefined` 的问题。
- 导出文件名在粘贴片段且无标题时默认使用 `片段排版_已排版.docx`。

## 使用方式

直接打开 `index.html` 即可使用。文件仅在本地浏览器中处理，不上传服务器。
